Hello World Server

Erstellen Sie einen Server, welcher immer "Hello World" ausgibt.

Nutzen Sie dafür https://nodejs.org/api/http.html

Starten Sie den Server mit der Console: node.exe [filename] oder via Webstorm.

require('http').createServer(function (req, res) {
    res.end('Hello World');
}).listen(3000);

console.log('Server running at http://localhost:3000/');

Hello World Server Version 2

Erstellen Sie einen Server, welcher immer die angefragte URL ausgibt. z.B. http://localhost:3000/Test5105 sollte Ihr Server folgendes ausgeben: You requested /Test5105.

const http = require('http');

const hostname = '127.0.0.1';
const port = 3000;

const server = http.createServer((req, res) => {
    res.writeHead(200, {'Content-Type': 'text/plain'});
    res.end('You requested ' + req.url);
});

server.listen(port, hostname, () => {
    console.log(`Server running at http://${hostname}:${port}/`);
});

Module 2: Callback-Übung

Erstellen Sie ein Module, wessen API ein Filename und einen Text erwartet.

  • Erstellen Sie dieses File mit dem übergebenem Text.
  • Lesen Sie danach das File.
  • Löschen Sie danach das File und geben Sie den gelesen Wert wieder zurück.
  • Behandeln Sie auch den Fehlerfall.
  • Nutzen Sie die API: https://nodejs.org/api/fs.html

Routing 1

Schreiben Sie einen Node-Server, welcher mit je nach URL eine eine Antwort erzeugt:

  • Route: /number
    • Senden die Zahlen 0 bis 50 zum Client.
    • Nutzen Sie dafür Module 1
    • Mit von und bis soll der Bereich definiert werden können z.B. /number?von=10&bis=50. Finden Sie die dafür geeignetste API-Funktionalität.
  • Route: /file
    • Beim Aufruf soll ein File erstellt werden mit dem aktuellen Datum. Dieser Wert soll ausgelesen werden und zum Client geschickt werden. Am Ende soll das File wieder gelöscht werden.
    • Nutzen Sie dafür Module 2
    • Mit 'fs' kann auf das File System zugegriffen werden: https://nodejs.org/api/fs.html.

Optional: Routing 2

Ergänzen Sie den Server von Routing 1 wie folgt:

Wenn "/to-send-html.html" angefragt wird, soll dessen HTML übertragen werden; dieselbe Funktionalität soll auch für das JavaScript File implementiert werden (to-send-js.js).

to-send-html.html und to-send-js.js befinden sich Ordner Vorlage.

Informationen zum Response-Objekt finden Sie unter https://nodejs.org/api/http.html#http_class_http_serverresponse

Eine Anfrage an "/to-send-html.html" kann so mit folgenden Schritten beantwortet werden:

  1. Laden Sie das HTML vom File-System mit 'fs'.
  2. Schreiben Sie den richtigen Content-Type in das Response-Object: res.writeHeader(200, {"Content-Type": "text/html"});
  3. Schreiben Sie den den Inhalt des html's in das Response-Object: res.write(html)
  4. Beenden Sie die Response: res.end();
  5. Verwenden Sie dieselbe Vorgehensweise für "/to-send-js.js"
  6. Vermeiden Sie Copy and Paste

Lösen Sie die Aufgabe mit der "FS Promises API".
